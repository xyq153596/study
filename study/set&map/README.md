# map -> 可以存放任意类型键值的集合
* Map.prototype.set() -> 设置键值，可以是任何值
* Map.prototype.get() -> 得到键值
* Map.prototype.clear() -> 清空Map
* Map.prototype.delete() -> 删除指定的元素。
* Map.prototype.entries() -> 返回一个迭代器。
* Map.prototype.has() -> 是否存在指定元素的键值。
* Map.prototype.keys() -> 返回一个key值的迭代器。
